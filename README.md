# Alien

## Test
To run all test cases
```
make test
```

## Build
To build the application.
```
make build
```

## Running the application
To run the application
```
./alien --map={inputfilename} --out={outputfilename} --n={numberofaliens}
```
Example:
```
./alien --map=in.txt --out=out.txt --n=2
```
